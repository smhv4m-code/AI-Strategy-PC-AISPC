# Data Flywheel Map

## Flywheel Loops

### Loop 1: Usage → Signal
**Score: 4/5**
*How does usage generate proprietary data?*
Daily enterprise usage creates rich, proprietary HR signals, it's gold and not public data, highly context driven

### Loop 2: Signal → Model
**Score: 2.5/5**
*How does that data improve the model?*

Potential is very high, but realized advantage is moderate. 
Candidate matching
Flight-risk prediction
Promotion readiness
Internal mobility recommendations
Skills inference
Workforce planning forecasts
HR case automation

### Loop 3: Model → Experience
**Score: 3/5**
*How does the better model improve UX?*
Better models can reduce friction:

Natural-language HR assistant
Smarter approvals
Better search
Personalized learning
Recruiting recommendations
Manager copilots
Auto-generated summaries

### Loop 4: Experience → Usage
**Score: 3.5/5**
*How does better UX drive more usage?*
HR tasks become faster and easier:
Managers self-serve more often
Employees use portals more frequently
Recruiters stay in-system longer
More workflows executed digitally
Higher adoption of optional module

**Total Flywheel Score: 13/20**
**Weakest Loop:*Loop 2: Signal → Model*
**Fix for weakest loop:

Unify fragmented HR data into one intelligence layer across recruiting, onboarding, performance, learning, and compensation so models have complete context instead of siloed signals.

Build specialized AI models for highest-value use cases such as candidate matching, attrition prediction, internal mobility, and succession readiness rather than one generic assistant.

Create continuous feedback learning loops using accepted recommendations, overrides, edits, and outcomes so models improve from real enterprise decisions over time.

## 90-Day Encroachment Plan

*Your partner played the Big Tech attacker. What was their plan to kill you?*

**Attacker:** OpenAI / Microsoft / AI-native HR startup

**Attack vector:** Launch an AI-first HR copilot that sits above SAP SuccessFactors and other systems, letting managers and employees hire, approve leave, answer policy questions, onboard staff, and manage talent through natural language—making the legacy UI optional.

**Your defense:** Turn SuccessFactors from system of record into system of intelligence by embedding native AI copilots inside workflows, leveraging proprietary HR data, superior compliance trust, deep integrations, and closed-loop learning that attackers cannot easily replicate.
